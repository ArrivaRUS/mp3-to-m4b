# mp3-to-m4b — концепты логотипа и базовой айдентики

> Шаг BrandDesigner (первый шаг фазы дизайна нового проекта). Здесь **4 РАЗНЫХ по идее**
> направления знака + вордмарка. Это **не финал** — финальный выбор делает человек (⛔ G2a),
> доводка выбранного направления (моно/иконка/фавикон, чистка) — после выбора.

## Бренд-бриф

- **Имя / вордмарк:** `mp3-to-m4b` (латиница). Тэглайн-подпись — латиница ASCII
  (`MP3 → M4B AUDIOBOOK BUILDER`), чтобы не зависеть от кириллических шрифтов в рендере
  (урок `research/m4b-toolchain.md` §7).
- **Суть (1–2 строки):** нативная macOS-утилита, что следит за папкой и собирает из mp3
  готовую аудиокнигу `.m4b` (главы + обложка) — с обязательным подтверждением качества/обложки.
- **Тон:** чистый, современный, «нативный macOS», тех-аккуратный (не игривый, не корпоративный).
- **Аудитория:** power user macOS («Антон» из PRD) — делает аудиокниги для себя, ценит контроль.
- **Ассоциации:** аудио · книга/главы · конвертация · надёжный системный инструмент.
- **Анти-референсы (чего избегать):** скевоморфные наушники/кассеты/винил; кислотные
  «музыкальные» неоны; мелкий мусор в иконке (не читается в 24–32 px); буквальная нотная
  символика (это аудиокниги, не музыка).

## Семейство с fb2-to-epub

Это «семья» с соседним приложением **fb2-to-epub** (тот же автор, тот же каркас watch-folder
converter). Я сознательно сохранил **общую визуальную грамматику** семьи, но дал mp3-to-m4b
**свой фирменный цвет**, чтобы две иконки читались как родственники и **различались в Dock**:

| Что | fb2-to-epub (сосед) | mp3-to-m4b (этот) |
|---|---|---|
| Тёмная панель + радиальный градиент + bloom-свечение + drop-shadow | да | **да (то же)** |
| Вордмарк | SF Pro Display bold, tracking −2, двухцветный (`fb2` светлый / `epub` градиент) | **то же (`mp3` светлый / `m4b` градиент)** |
| Морф-стрелка `→` со «спарком» между частями имени | да | **да (стрелка; спарк убран — у нас play/waveform)** |
| Фирменный градиент | **тёплый**: orange `#FFB23D` → red `#FF3D5A` → magenta `#E63CC8` («текст/страница») | **холодный «аудио»**: cyan `#34E0D2` → teal `#22B5E0` → indigo `#4A6BFF` |
| Мотив знака | страница/книга + спарк | **аудио** (waveform / play / главы / монограмма) |

**Развилка для человека (ответить на согласовании):**
- **(A) В одном семействе с соседом** — да, это все 4 направления (общий каркас, новый
  «аудио»-цвет). Рекомендация BrandDesigner: для пары утилит одного автора так чище.
- **(B) Полностью самостоятельный знак** — если хочется, чтобы mp3-to-m4b не выглядел «братом»
  fb2 (напр. другой силуэт панели, другой характер). Тогда доводим выбранное направление в
  сторону бóльшей самостоятельности. Скажи, если этот вариант интереснее.

## Общая палитра (питает токены) — едина для всех направлений

Тёмное приложение (как у соседа). Фирменный градиент работает на тёмном; для светлого фона
вордмарка достаточно (текст тёмный) — см. контактный лист (он рендерится Юркой).

| Роль | HEX | Назначение |
|---|---|---|
| **Brand / accent 1 (cyan)** | `#34E0D2` | начало фирменного градиента, hover/active, play-акцент |
| **Brand / accent 2 (teal)** | `#22B5E0` | середина градиента, основной «аудио»-тон |
| **Brand / accent 3 (indigo)** | `#4A6BFF` | конец градиента, ссылки/прогресс |
| Panel BG (центр) | `#14202A` | фон тёмной панели (радиальный, центр) |
| Panel BG (край) | `#070B10` | фон тёмной панели (радиальный, угол) — гарантирует тёмные углы иконки |
| Edge glow | `#136B86` | свечение по краю панели (низкая прозрачность) |
| Text high | `#EAF6FA` | основной текст / `mp3` в вордмарке |
| Text muted | `#6E8390` | микро-подпись, второстепенный текст |
| On-accent / sheen | `#EAFBFF` | play-треугольник, блик на знаке |

Градиент-стоп (для токена `brand.gradient`): `#34E0D2 → #22B5E0 → #4A6BFF` (диагональ ~115°).

## Типографика (питает токены)

- **Вордмарк / заголовки:** **SF Pro Display / SF Pro Text** (системный `-apple-system`,
  fallback `Helvetica, Arial`), **Bold (700)**, tracking `−2` для крупного начертания.
  Это «нативный macOS» характер и 1-в-1 с соседом.
- **Текст интерфейса:** SF Pro Text Regular/Medium (системный стек).
- **Микро-подпись (caps-label):** тот же стек, 600, **tracking +6**, цвет `Text muted`.
- **Запасные обложки (cairosvg, отдельная задача F6):** кириллица — только «зелёный» список
  из research §7 (Georgia / Helvetica / Gill Sans / Palatino / Baskerville-roman / Hoefler Text);
  **не** Didot/Futura/Avenir. К логотипу не относится (он латиницей), но фиксирую, т.к. это
  единый бренд-стек проекта.

## Направление 01 — Waveform (звуковая волна)

**Файлы:** `01-waveform-icon.svg`, `01-waveform-lockup.svg`
**Идея:** симметричная звуковая волна из скруглённых столбиков; центральный (самый высокий) —
ярче, с бликом, задаёт «play-вперёд» движение. Самый прямой «аудио»-знак.

- **Палитра:** общая (cyan→teal→indigo), центральный бар — отдельный светлый градиент.
- **Шрифт вордмарка:** SF Pro Display Bold, tracking −2.
- **Плюсы:** мгновенно читается как «звук/аудио»; **лучше всех держится в 24 px** (фавикон-тест
  пройден — волна узнаваема); чистый, спокойный, очень «нативный».
- **Минусы:** waveform — частый приём для аудио-приложений (меньше «уникальности»); сам по себе
  не говорит «**книга/главы**» (говорит «аудио вообще»).
- **Семейное / самостоятельное:** семейное по каркасу и цвету; силуэт знака — свой (у соседа
  страница, тут волна).

## Направление 02 — Книга + play (audiobook)

**Файлы:** `02-book-play-icon.svg`, `02-book-play-lockup.svg`
**Идея:** раскрытая книга-лист (прямой привет каркасу семьи fb2), но из корешка вырастает не
спарк, а **play-треугольник** — «книга, которую слушаешь». Аудиокнига одним знаком.

- **Палитра:** общая; play-треугольник — светлый (`#EAFBFF`) со свечением.
- **Шрифт вордмарка:** SF Pro Display Bold, tracking −2.
- **Плюсы:** точнее всех передаёт смысл «**аудиокнига**» (книга + play); **сильнейшая
  семейная связь** с fb2 (тот же жест книги-листа); хорошо читается в 24 px.
- **Минусы:** play частично ложится на строки правой страницы (в доводке развести/уменьшить
  блик); «книга + play» — тоже узнаваемый штамп жанра.
- **Семейное / самостоятельное:** **максимально семейное** (тот же мотив книги, что у соседа,
  но в «аудио»-цвете и с play).

## Направление 03 — Главы-сегменты (chapters → one book)

**Файлы:** `03-chapters-icon.svg`, `03-chapters-lockup.svg`
**Идея:** левый столбец отдельных скруглённых сегментов (исходные mp3 = главы) через
коннекторы **сливается в один высокий корешок** (готовый `.m4b`) с play-вырезом. Говорит про
суть продукта: «много файлов → одна книга с главами».

- **Палитра:** общая; собранный корешок — отдельный вертикальный градиент.
- **Шрифт вордмарка:** SF Pro Display Bold, tracking −2.
- **Плюсы:** концептуально **самый «про продукт»** (главы + сборка-конвертация — то, чего нет
  у соседа); хорошо смотрится в крупном/среднем размере.
- **Минусы:** **самый busy**; в 24 px сегменты сливаются в «штрих-код/список», метафора теряется
  (фавикон-тест слабый); play-вырез в доводке надо сделать чище (сейчас чуть «полый»).
- **Семейное / самостоятельное:** скорее **самостоятельное** по идее (у соседа нет «глав»);
  каркас/цвет — семейные.

## Направление 04 — Монограмма «4» / M4B

**Файлы:** `04-monogram-icon.svg`, `04-monogram-lockup.svg`
**Идея:** жирная геометрическая **«4»** (ядро имени m**4**b) с **play-треугольником** в
открытом контуре цифры и тонкой **waveform-подложкой**. Самый «брендовый»/буквенный путь.

- **Палитра:** общая; «4» — фирменный градиент, play — светлый, waveform — горизонтальный
  градиент.
- **Шрифт вордмарка:** SF Pro Display Bold, tracking −2 (монограмма нарисована вектором, не текст).
- **Плюсы:** самый **отличный от fb2** (сосед — символьный, не буквенный); сильная, лаконичная
  «4» — узнаётся даже в 24 px; легко переносится в моно/один цвет.
- **Минусы:** «4» сама по себе не кричит «аудиокнига» (смысл несут play+waveform, а они в
  маленьком размере теряются — в доводке waveform-подложку упростить/убрать для small-иконки);
  риск «технологичности» в ущерб теплоте.
- **Семейное / самостоятельное:** **самое самостоятельное** (другой класс знака — монограмма);
  держит семью только цветом и вордмарком.

## Рекомендация BrandDesigner (не решение — решает человек)

- Если важнее всего «**это аудиокниги**» и связь с соседом → **02 (книга + play)**.
- Если важнее «чисто, узнаваемо, лучший small-size» → **01 (waveform)**.
- Если хочется подчеркнуть «**это про главы/сборку**» (уникальность продукта) → **03**, но
  с доводкой под мелкий размер.
- Если хочется отстроиться от fb2 и иметь сильный буквенный знак → **04**.

Мой топ-2 на доводку: **02** (смысл + семья) и **01** (чистота + фавикон). Финал — за человеком.

## Тех-заметки по рендеру (для доводки)

- Все 8 SVG **валидны и проверены рендером** (headless Chrome → PNG, + small-size 48/24 px).
  Знаки видимые, не «невидимые»; монограмма с `fill-rule="evenodd"` рендерится корректно.
- Везде **сплошные заливки** (без тонких штрихов под фильтрами) — по уроку
  `cairosvg-thin-stroke` / research §7. Свечение/тени — поверх сплошных форм.
- Иконки нарисованы **full-bleed на 1024×1024** (тёмные углы гарантированы сплошной заливкой),
  как app-icon у соседа на Tahoe — **система сама скругляет углы**; в контактном листе Юрки
  показывать с macOS-скруглением (`border-radius ≈ 22%`).
- Известный полиш-нюанс на доводку: верхний «highlight rim» (как у fb2) на нашем более тёмном
  фоне читается лёгкой дугой-«бровью» в крупном размере — в финале приглушить/перерисовать
  (в 24–48 px не виден, на выбор направления не влияет).
