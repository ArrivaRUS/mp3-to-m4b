## §M0 — Протокол (fake-engine), РИСК №1
Тесты (Tester unit/integration + Юрка прогон):
- **I2:** нет файла-команды в `queue/commands/` → книга НИКОГДА не уходит в `converting` (нет сборки).
- **Stale-rev:** команда с устаревшим `source_rev` (файлы книги изменились) → `confirm_rejected_stale`, без сборки.
- **Идемпотентность дубля:** две команды с одним `idempotency_key`/двойной клик → ровно один build.
- **Malformed:** битый JSON в `queue/commands/` → уезжает в `commands/bad/`, агент не падает, сборки нет.
- **Прерванное `converting`:** рестарт агента без живого pid → `error: interrupted` + temp вычищены.
- **Rising-edge:** появление `pending-confirm` поднимает окно (закрытое — открывается).
- **I1:** исходные файлы не меняются/не удаляются. **G5:** свежий выход → повторно не пересобирать.
- **Один писатель:** в коде приложения НЕТ записи в `state.json`/`queue/books/*` (ревью-проверка).

## §M0.5 — Разведка
- Теги полные → автор←artist/album, название←album, порядок←track, имена глав←title.
- Теги пустые, папка «Толстой - Война и мир» → парс автор/название; имена глав из имён файлов.
- Натуральная сортировка `01,02,…,10` (не `1,10,2`). Кириллица не ломается.
- Детект встроенной обложки (APIC) → превью из первого файла с ней.

## §M1 — Движок и вертикаль (по research)
- N mp3 (разные SR/каналы) → один `.m4b`, brand `M4A`, faststart; длительность = сумма **без дрейфа** (concat+aformat).
- `ffprobe -show_chapters`: N глав, верные START/END (1/1000), кириллические имена, fallback-имена.
- Обложка attached_pic (mjpeg) присутствует; исходники целы (I1); атомарность (нет полу-`.m4b` при сбое/отмене).
- **Цепочка обложки:** из mp3 / нет→веб (квадрат-кандидаты) / нет сети→генерация Pillow (квадрат, кириллица ок, не tofu) / ручная замена. Пустых обложек нет (G4-цель PRD).
- **Нарезка** (P1): части по границам глав, каждая валидна, своя обложка, `track=N/M`, `-map_chapters 1` (нет дублей/фантомных глав); порог<главы → часть=глава + предупреждение.

## §M1 — Edge-cases (E1–E18 PRD §9)
Нет тегов · разные SR/каналы · битый mp3 (warn «без него») · нет интернета (→генерация) · нет места
(danger, cleanup) · повторная конвертация (идемпотентность) · отмена в процессе · очень длинная книга
(лимит дескрипторов→fallback demuxer) · мусор в папке · недокопированный mp3 · недопустимые символы в
имени · один mp3 в подпапке · дубли имён глав · размонтирование · порог<главы · краш при сборке (cleanup).
